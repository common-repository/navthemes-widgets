== Description ==

This plugin adds a NavThemes Widget in your Widgets ares. Comes with Heading, Content, image, logo and Button text and links. This can be used to showcase anything informative. Its compatible NavThemes all Themes and standard themes. 

== Installation ==
 
1. Upload `remove-website-url-from-comment-form.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Thats it. Now Comment form will not show website url.
 
== Frequently Asked Questions ==
 
= Should i take my website backup before activating this plugin ? =

This plugin is real simlple and does not do anything which can break your site but its always safe to take a backup whenever you doing anything with plugins.
